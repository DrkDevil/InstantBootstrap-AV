What is InstantBootstrap?

InstantBootstrap is an Express Setup Environment.


InstantBootstrap Angular Edition was created to decrease environment setup time when wanting to create new WebApps built with bootstrap. We wanted a quick way to be able to have all our commonly used dependencies and favorite plug-ins ready to go with minimal commands in terminal. InstantBootstrap Delivers that exactly that with a preset environment for bootstrap sass and angular development. This will help you get your projects started faster, keeping you more efficient and productive.
